# Synthetic Spirit

**Status:** Experimental  
**Owner:** Alyssa Heeter  
**Classification:** EXPERIMENTAL

---

## Purpose

Synthetic Spirit is an experimental frontend sandbox used for prototyping UI concepts and design systems — primarily for the CeceLaw ecosystem.

## What it solves

Provides a low-stakes environment to test layouts, components, and visual concepts before integrating them into production projects like `cece-law-website-v2`.

## Tech Stack

- HTML / CSS / JavaScript
- Vite (build tooling)

## Structure

```
/cece-law    ← CeceLaw UI prototype experiments
```

## Status

Experimental — not a production system. Concepts that graduate go into `cece-law-website-v2`.
